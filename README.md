# Portfolio
**O mnie:**

  z zamiłowaniem do testowania, gdzie staram się odnaleźć swoje miejsce.
  Póki co z doświadczeniem stażysty dla podmiotu realizującego zlecenia dla grupy z branży paliwowej.
